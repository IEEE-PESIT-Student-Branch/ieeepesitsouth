3D Printing Workshop


Date : 17th March, 2018

Venue : PES Institute of Technology - Bangalore South Campus

The workshop on "3D Printing, 3D Scanning and 3D Designing" enables you to learn from a beginner level to an advanced user level in the field of 3D Printing, 3D Scanning and 3D Designing, the knowledge which you can add to your skill set and apply in any profession worldwide. Come improve your knowledge and become a part of the3rd Industrial Revolution-3D Printing.

3D Printing

  1. Introduction to 3D Printing
  2. History of 3D Printing
  3. Types of 3D Printing technologies
  4. Applications of 3D Printing
  5. Materials used in 3D Printing
  6. Demonstration of the FDM 3D Printing process
  7. Impact of 3D Printing across the world
  8. Future of 3D printing
  9. Career aspects in 3D Printing
  10. 4D printing

Slicing

  1. Introduction to the Slicing process
  2. Slicing software installation
  3. .STL files and G code files
  4. Slicing .STL files
  5. Parameters in Slicing : layer height, overhangs, supports, raft, brim, extrusion temperature, preheating, shells, etc.

3D Scanning

  1. Introduction to 3D Scanning
  2. Types of 3D Scanning technologies
  3. Applications of 3D Scanning
  4. Comparison between the types of 3D Scanning technologies
  5. 3D Scanning demonstration

3D Designing/3D Modeling

  1. Introduction to 3D Modeling
  2. Types of 3D Modeling
  3. 3D Modeling for 3D Printing
  4. 3D Modeling software installation : CAD Modeling and Organic Modeling
  5. 3D Modeling using different software
  6. Applications of 3D Modeling

System Requirements

  • 1. Organic modeling software: Sculptris (Compatible with Windows and Mac.)
  • Windows
  • Operating System: Windows 7 and above (64-bit)
  • Processor: 1Ghz Pentium 4 and above (or compatible).
  • Memory: 1 GB of RAM.
  • Graphic Card: ATI or NVidia card with openGL 2.0 support.
  • Screen Resolution: 1024x768
  • Recommended: Wacom or compatible graphics tablet

  • Mac OSX
  • Operating System: Mac OS X 10.5/10.6.
  • Processor: 2Ghz core 2 duo and above (or compatible).
  • Memory: 1 GB of RAM.
  • Graphic Card: ATI or NVidia card with openGL 2.0 support.
  • Screen Resolution: 1024x768
  • Recommended: Wacom or compatible graphics tablet


  • 2. CAD modeling software: Tinker CAD (Online)

    • Tinkercad is an easy, browser-based 3D design and modeling tool for all. Tinkercad allows users to imagine anything and then design it in minutes. Used by millions of Tinkerers across the world, it is a friendly software where one can create powerful 3D designs and export it for 3D printing or laser cutting.

  • Windows
  • Windows 7 and above (64-bit)
  • Intel® Pentium® 4 or AMD Athlon® 64 or later with 2 GHz or faster processor; or compatible
  • 2+ GB RAM (1.5 GB minimum)
  • 1.5+ GB free disk space (for installation)
  • Direct3D® 9 or 10 graphics support with 64+ MB

  • Mac OSX
  • Apple® Mac OS® X, version 10.7 or later with 64-bit Intel processor
  • Apple® Mac® Pro 4.1 or later; MacBook® Pro 5.1 or later (MacBook Pro 6.1 or later recommended); iMac® 8.1 or later (iMac 11.1 or later recommended)
  • 3 GB of RAM (4 GB recommended)
  • 2.5 GB free disk space for download and installation (3 GB recommended)
  • 1280×800 display with true color (1600×1200 with true color recommended)
  • All Mac OS X supported operating system languages


  • 3. Slicing software: Craftware (Compatible with Windows and Mac.)
  • Windows : 7 and above
  • Mac OSX : 10.7 and above
