Micro Workshop on BASH Scripting

Date : 5th March, 2018

IAll participants must bring one laptop with at least one of the following configurations:-.

  1. Linux Distribution, on main or dual boot
  2. Linux Distro on VM on Windows
  3. Cygwin on Windows
  4. MacOS with Shell Switched to BASH

Pre-requisites:-

  1. Elementary knowledge about the GNOME terminal or any command line interface
  2. Fundamental programming (C/C++/Java/Python)
  3. Basics of computer hardware
